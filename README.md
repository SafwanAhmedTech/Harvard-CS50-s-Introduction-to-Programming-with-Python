CS50P — Introduction to Programming with Python

This repository contains my solutions, exercises, labs, and final project completed during Harvard University's CS50P: Introduction to Programming with Python course.

CS50P is an introduction to programming using Python, designed to teach the fundamentals of software development through hands-on problem solving and practical projects.

📚 Course Overview

The course focuses on building strong programming foundations and learning how to write clear, efficient, and reliable Python programs.
